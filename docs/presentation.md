
# Présentation du projet Blog Statique

## Description

Blog Statique est un site web statique conçu comme exercice pratique pour maîtriser les workflows Git (branches, commits, merges, pull requests). Le contenu du blog porte sur le développement web et les technologies modernes.

## Objectifs

- Pratiquer les commandes Git dans un contexte de projet réel
- Comprendre le cycle de vie d'un projet web (développement, déploiement)
- Créer un blog fonctionnel avec HTML, CSS et scripts shell

## Structure du projet

```
blog_statique/
├── index.html              # Page d'accueil
```

## Technologies utilisées

- **HTML5** — Structure des pages
- **CSS3** — Mise en forme et responsive design
- **Markdown** — Documentation
- **Git** — Gestion de versions
- **Apache** — Serveur web pour le déploiement
