Apache : Serveur Web

Apprenez à installer et configurer Apache sur Ubuntu pour héberger vos sites.

Qu'est-ce qu'Apache ?

Apache HTTP Server est l'un des serveurs web les plus utilisés au monde depuis plus de 25 ans. Son rôle est simple : recevoir des requêtes HTTP d'un navigateur et renvoyer les fichiers correspondants (HTML, CSS, images). C'est lui qui rend votre site accessible dans un navigateur via l'adresse http://localhost.

Pourquoi utiliser Apache ?

  • Gratuit et open source
  • Très stable et éprouvé depuis 1995
  • Compatible avec tous les systèmes Linux
  • Facile à configurer pour les débutants
  • Supporte les virtual hosts (plusieurs sites sur un serveur)

Installation sur Ubuntu / Debian

Ouvrez un terminal et tapez les commandes suivantes une par une :

  • sudo apt update — mettre à jour la liste des paquets
  • sudo apt install apache2 -y — installer Apache
  • sudo systemctl start apache2 — démarrer le service
  • sudo systemctl enable apache2 — démarrage automatique
  • sudo systemctl status apache2 — vérifier l'état

Tester l'installation

Une fois Apache installé, ouvrez votre navigateur et tapez http://localhost. Vous devriez voir la page d'accueil par défaut d'Apache avec le message "It works!". Si c'est le cas, Apache fonctionne correctement sur votre machine.

Déployer son site

Par défaut, Apache sert les fichiers depuis le dossier /var/www/html/. Pour déployer votre blog, copiez vos fichiers dans ce dossier :

  • sudo cp -r * /var/www/html/ — copier les fichiers
  • sudo chmod -R 755 /var/www/html/ — donner les droits
  • Ouvrez http://localhost pour voir votre site

Conclusion

Apache est un outil fondamental pour tout développeur ou administrateur système. Combiné à Linux et Git, il forme le trio parfait pour déployer des projets web professionnels rapidement et efficacement.
