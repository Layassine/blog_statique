Déployer un site avec Apache

Publié le 22 janvier 2025 — par Hamza Aourik

Qu'est-ce qu'Apache ?

Apache HTTP Server est l'un des serveurs web les plus utilisés au monde. Il répond aux requêtes HTTP et affiche votre site dans le navigateur.

Installation sur Ubuntu

  • sudo apt update
  • sudo apt install apache2
  • sudo systemctl start apache2
  • sudo systemctl enable apache2

Déployer son site

Copiez vos fichiers HTML dans /var/www/html/ pour que le site soit accessible via http://localhost.

Vérifier le déploiement

Ouvrez votre navigateur et tapez http://localhost. Si votre site s'affiche, c'est réussi ! Vous pouvez aussi accéder via http://VOTRE_IP depuis une autre machine.

Conclusion

Apache est simple à installer et fiable pour héberger des sites statiques. C'est un excellent point de départ pour comprendre les serveurs web.
