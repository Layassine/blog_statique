Introduction à Linux

Publié le 15 janvier 2025 — par Hamza Aourik

Qu'est-ce que Linux ?

Linux est un système d'exploitation open source créé par Linus Torvalds en 1991. Il est aujourd'hui utilisé sur des millions de serveurs à travers le monde.

Pourquoi apprendre Linux ?

La majorité des serveurs web et des environnements de développement professionnels tournent sous Linux. Le maîtriser est un atout majeur pour tout développeur ou administrateur système.

Les commandes de base

  • ls — lister les fichiers d'un dossier
  • cd — changer de répertoire
  • mkdir — créer un dossier
  • cp — copier un fichier
  • rm — supprimer un fichier
  • sudo — exécuter en mode administrateur

Conclusion

Linux est un outil puissant. En pratiquant régulièrement dans un terminal, vous serez rapidement à l'aise avec cet environnement.
