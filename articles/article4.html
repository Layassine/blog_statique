HTML & CSS : Les bases du web

Comprendre les deux langages fondamentaux pour créer des pages web.

Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage de base de toutes les pages web. Il définit la structure et le contenu d'une page : titres, paragraphes, images, liens, tableaux, formulaires. Sans HTML, il n'y a pas de page web.

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est le langage qui gère l'apparence visuelle des pages HTML. Il contrôle les couleurs, les polices, les tailles, les marges, les animations et la mise en page. Un site peut fonctionner sans CSS, mais il sera très basique visuellement.

Les balises HTML essentielles

  • <html> — racine de la page
  • <head> — métadonnées (titre, CSS, charset)
  • <body> — contenu visible de la page
  • <h1> à <h6> — titres de différents niveaux
  • <p> — paragraphe de texte
  • <a href=""> — lien hypertexte
  • <img src=""> — image
  • <div> — bloc conteneur générique
  • <header>, <main>, <footer> — balises sémantiques

Les propriétés CSS essentielles

  • color — couleur du texte
  • background-color — couleur de fond
  • font-size — taille de la police
  • margin / padding — espacement extérieur / intérieur
  • display: flex — mise en page flexible
  • display: grid — mise en page en grille
  • border-radius — coins arrondis
  • box-shadow — ombre portée

La règle d'or

HTML s'occupe du fond (contenu, structure, sémantique) et CSS s'occupe de la forme (couleurs, tailles, mise en page). Ne mélangez jamais les deux : évitez les styles inline dans le HTML, centralisez tout dans un fichier CSS séparé.

Conclusion

HTML et CSS sont les fondations incontournables du développement web. Maîtrisez-les avant de passer à JavaScript ou aux frameworks. La pratique régulière est la clé : recréez des sites existants pour progresser rapidement.
