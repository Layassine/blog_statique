Les bases du HTML et CSS

Publié le 5 février 2025 — par Hamza Aourik

Le HTML : la structure

HTML (HyperText Markup Language) est le langage de base du web. Il définit la structure d'une page grâce à des balises.

  • <h1> — titre principal
  • <p> — paragraphe
  • <a href="..."> — lien
  • <img src="..."> — image
  • <div> — conteneur générique

Le CSS : le design

CSS (Cascading Style Sheets) contrôle l'apparence visuelle des éléments HTML : couleurs, tailles, marges, polices.

  • color — couleur du texte
  • background-color — couleur de fond
  • margin et padding — espacement
  • font-size — taille du texte
  • display: flex — alignement des éléments

Conclusion

HTML et CSS sont les fondements de tout développement web. Les maîtriser permet de créer des interfaces propres et accessibles.
