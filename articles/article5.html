Introduction à Docker

Découvrez la conteneurisation et pourquoi Docker révolutionne le déploiement.

Qu'est-ce que Docker ?

Docker est une plateforme de conteneurisation lancée en 2013. Elle permet d'empaqueter une application et toutes ses dépendances dans un conteneur — une unité légère et portable qui fonctionne de manière identique sur n'importe quelle machine, que ce soit votre ordinateur personnel, un serveur de production ou le cloud.

Conteneur vs Machine Virtuelle

Beaucoup confondent les deux. Voici la différence essentielle :

  • Machine Virtuelle — émule un ordinateur complet avec son propre OS (lourd, lent)
  • Conteneur Docker — partage le noyau du système hôte (léger, rapide)
  • Un conteneur démarre en quelques secondes, une VM en plusieurs minutes
  • Un conteneur utilise beaucoup moins de mémoire qu'une VM

Les concepts clés

  • Image — le modèle de base (comme une recette de cuisine)
  • Conteneur — une instance en cours d'exécution d'une image
  • Dockerfile — le fichier qui décrit comment construire une image
  • Docker Hub — le registre public d'images Docker
  • docker-compose — outil pour gérer plusieurs conteneurs ensemble

Commandes Docker essentielles

  • docker pull nginx — télécharger une image depuis Docker Hub
  • docker run -d -p 80:80 nginx — lancer un conteneur
  • docker ps — lister les conteneurs actifs
  • docker stop id — arrêter un conteneur
  • docker build -t monapp . — construire une image depuis un Dockerfile
  • docker images — lister les images locales

Docker dans notre projet

Pour notre blog statique, Docker permet de déployer Apache et notre site dans un conteneur portable. Voici un exemple simple de Dockerfile :

  • FROM httpd:2.4 — utiliser l'image Apache officielle
  • COPY . /usr/local/apache2/htdocs/ — copier les fichiers du site
  • EXPOSE 80 — exposer le port 80

Conclusion

Docker simplifie radicalement le déploiement d'applications. Avec un seul fichier Dockerfile, n'importe qui peut lancer votre projet en une commande, sans se soucier de la configuration du serveur. C'est aujourd'hui une compétence très recherchée dans le monde professionnel.
