Introduction à Docker

Publié le 12 février 2025 — par Hamza Aourik

Qu'est-ce que Docker ?

Docker est un outil de conteneurisation. Il permet d'empaqueter une application avec tout ce dont elle a besoin pour fonctionner, et de la faire tourner n'importe où de façon identique.

Pourquoi utiliser Docker ?

Docker résout le problème classique "ça marche sur ma machine mais pas sur la tienne". En conteneurisant l'application, l'environnement est identique partout.

Commandes Docker de base

  • docker pull image — télécharger une image
  • docker run image — lancer un conteneur
  • docker ps — lister les conteneurs actifs
  • docker stop id — arrêter un conteneur
  • docker build -t nom . — construire une image

Conclusion

Docker est devenu incontournable dans les équipes de développement modernes. Le comprendre, même en surface, est un vrai plus sur un CV.
