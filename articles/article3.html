Travailler avec Git et GitHub

Publié le 28 janvier 2025 — par Hamza Aourik

Qu'est-ce que Git ?

Git est un logiciel de gestion de versions. Il permet de suivre l'historique de chaque modification d'un projet et de travailler en équipe sans se marcher dessus.

Les commandes Git essentielles

  • git init — initialiser un dépôt Git
  • git clone URL — copier un dépôt existant
  • git status — voir les fichiers modifiés
  • git add . — préparer tous les fichiers
  • git commit -m "message" — enregistrer les modifications
  • git push — envoyer sur GitHub
  • git pull — récupérer les dernières modifications

Travailler avec des branches

Une branche permet de travailler sur une fonctionnalité sans affecter le code principal.

  • git checkout -b ma-branche — créer et basculer sur une branche
  • git checkout main — revenir sur la branche principale
  • git merge ma-branche — fusionner une branche

Conclusion

Git est indispensable dans tout projet professionnel. Utiliser GitHub en équipe avec des branches et des pull requests est une pratique standard dans l'industrie.
